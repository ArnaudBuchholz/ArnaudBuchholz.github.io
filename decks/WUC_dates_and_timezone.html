Good to know * new Date() => now * new Date(1671926400000) => Xmas ad midnight UTC (using Unix) * new Date('2022-12-25') => Xmas at midnight UTC ==> POSSIBLY NOT THE RIGHT DATE !!! * new Date('2022-12-25T00:00:00.000Z') => Xmas at midnight UTC * new Date('2022-12-25 00:00:00') => Xmas at local UTC
Advices * If time is relevant, make sure to store the information in UTC * If you want to manipulate a date only, make sure to to initialize the date object appropriately
