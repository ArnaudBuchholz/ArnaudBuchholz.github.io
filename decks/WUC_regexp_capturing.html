Regular expressions are only for matching strings
is a common misconception.


It is a powerful too that can also extract values from the matched string.


Regular expressions and capturing groups are tightly integrated with the JavaScript language.

What is a capturing group ?

A pair of parenthesis* define a capturing group. 

const imageDescription = 'Image resolution is 1440×900 pixels.';
const regexpSize = /([0-9]+)x([0-9]+)/;
const match = imageDescription.match(regexpSize);
console.log(`Width: ${match[1]} / Height: ${match[2]}.`);
// expected output: "Width: 1440 / Height: 900."

NOTE : to capture a repeated pattern, make sure the repetition is enclosed inside parenthesis or the regexp captures only the last occurrence.


* The specific syntax (?: ) defines a non-capturing group.

The result of a match is an array-like where : 

[
  '1440x900',
  '1440',
  '900',
  index: 31,
  input: 'This image has a resolution of 1440x900 pixels.'
]
  • match[0] contains the matching string,
  • match[1] contains the value of the first capturing group,
  • match[2] contains the value of the second capturing group,
  • ...

string.replace

Used with a replacer function, the capturing group values are provided as additional parameters. 


// 
Title
 => 
Title


html.replace(/<(h\d)>([^<]*)<\/h\d>/ig, (match, titleTag, content) =>
  `<div class="${titleTag}">${content}</div>`
);

Regular expression explanation and matching example

string.split

The resulting array
includes the capturing group values. 

// Extracting all vowels from a string

"Arnaud was here".split(/[aeiouy]+/i); // no capturing group
// [ '', 'rn', 'd w', 's h', 'r', '' ]

"Arnaud was here".split(/([aeiouy]+)/i); // captures vowels
// [ '', 'A', 'rn', 'au', 'd w', 'a', 's h', 'e', 'r', 'e', '']

"Arnaud was here".split(/([aeiouy]+)/i).filter((_, pos) => pos % 2);
// [ 'A', 'au', 'a', 'e', 'e' ] ✌
