GPF Library

A multi-purpose JavaScript library created and maintained by Arnaud Buchholz.

Plato analysis

Features

  • Compatible with several hosts (cscript/wscript, NodeJS, Rhino, PhantomJS, most recent browsers)
  • Namespace and class system -- not implemented yet
  • Java-like annotation tool (attributes) -- not implemented yet
  • Interface based -- not implemented yet
  • Asynchronous binary / textual streams -- not implemented yet
  • XML-aware -- not implemented yet
  • Self-Tested

This library is developed in conjunction with a blog where the concepts and algorithms are documented and explained.
