Tutorial: LOADING

LOADING

Loading GPF Library

cscript/wscript

Using Windows Script File format

<?xml version="1.0"?>
<job id="gpfExample">
  <script language="JScript" src="gpf.js" />
  <script language="JScript">
    // use gpf namespace
  </script>
</job>

Using Scripting.FileSystemObject object

var fso = new ActiveXObject("Scripting.FileSystemObject");
eval(fso.OpenTextFile("gpf.js", 1, false, 0).ReadAll());
// use gpf namespace

NodeJS

Will be modified when the NPM package will be ready

var gpf = require("./gpf.js");

Rhino

Use load

load("./gpf.js");
// use gpf namespace

PhantomJS

require("./gpf.js");
// use gpf namespace

or

<script src="gpf.js"></script>
<script>
    // use gpf namespace
</script>

Browsers

<script src="gpf.js"></script>
<script>
    // use gpf namespace
</script>
