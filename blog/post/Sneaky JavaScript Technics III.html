Sneaky JavaScript Technics III

A ninja is a master of disguise: how do you hide an object member so that nobody can use it? Here are several ways.
### The context These days, I am starting a new task on my [gpf-js project](https://github.com/ArnaudBuchholz/gpf-js) and it implies manipulating an [AST](https://en.wikipedia.org/wiki/Abstract_syntax_tree) structure within [NodeJS](https://nodejs.org/en/). Because the structure analysis requires to have access to the parent nodes, the AST items are augmented with a link to their parent.
