My own jsdoc plugin

Jsdoc provides a convenient way to document APIs by adding comments to the code. However, the task can be really tedious when it comes to documenting every details of each symbol. Luckily, the tool provides ways to interact with the parsing process through the creation of plugins.
## Introduction Just before the release of version 0.1.5, I started to focus on documentation. I knew I had to use [jsdoc](http://usejsdoc.org/) but I started experimenting very late. I soon realized that I had to do lot of copy & paste... The lazy developer in me started to get annoyed and I started to look at ways I could do things more efficiently. The official jsdoc documentation page about plugin is quite short...
